SQL Data Exploration &
Tableau Dashboard
for Player Statistics in English Premier League 2018-2019 season

Analyzing the player statistics who were in 2018-2019 EPL season in SQL in order to create an interactive Tableau dashboard.

Data Collection and Manipulation

Data was collected from Kaggle. There were 43 columns.

Columns =['full_name', 'age', 'birthday', 'birthday_GMT', 'league', 'season', 'position', 'Current Club', 'minutes_played_overall', 'minutes_played_home', 'minutes_played_away', 'nationality', 'appearances_overall', 'appearances_home', 'appearances_away', 'goals_overall', 'goals_home', 'goals_away', 'assists_overall', 'assists_home', 'assists_away', 'penalty_goals', 'penalty_misses', 'clean_sheets_overall', 'clean_sheets_home', 'clean_sheets_away', 'conceded_overall', 'conceded_home', 'conceded_away', 'yellow_cards_overall', 'red_cards_overall', 'goals_involved_per_90_overall', 'assists_per_90_overall', 'goals_per_90_overall', 'goals_per_90_home', 'goals_per_90_away', 'min_per_goal_overall', 'conceded_per_90_overall', 'min_per_conceded_overall', 'min_per_match', 'min_per_card_overall', 'min_per_assist_overall', 'cards_per_90_overall']

Of which 17 columns were derived from other columns. 12 columns of these Columns=[ 'goals_involved_per_90_overall', 'assists_per_90_overall', 'goals_per_90_overall', 'goals_per_90_home', 'goals_per_90_away', 'min_per_goal_overall', 'conceded_per_90_overall', 'min_per_conceded_overall', 'min_per_match', 'min_per_card_overall', 'min_per_assist_overall', 'cards_per_90_overall'] were deleted and were then reentered using SQL queries and formulas

Birthday and Birthday_GMT

Birthday and Birthday_GMT column are not necessary for this particular data analysis as the age of the players are given.

Data Exploration in SQL

  • Bench_strength for different positions in each club
  • Showing the nationality of players
  • Top ten fair players with minutes_played_overall
  • Top goal scorers in the league
  • Age and Experience of players in the league
  • Top players with more assists and more cleansheets

Only first four columns were considered for data visualisation

Tableau Dashboard

Data was then imported into Tableau for the creation of the dashboard.

Table Representing Top Goal Scorers In The League

Table was created using goals_scored_overall column against the name of player. Top 10 goal scores were used for the creation of the chart.

World Map Representing Nationality of Players

Map was created using the Nationality of Players, with the Count of Players of same nationality.

Stacked Bar Chart Representing Bench Strength of Players by Position In Each Team

Stacked bar chart was created using Count of Players, considering Position column, within each team.

Horizontal Bar Chart Representing The Top 10 Fair Players In The League

Stacked bar chart was created using Count of Players, considering derived column " Minutes_per_card_overall" and "Cards_overall" column. The ones with max minutes_per_card ratio and less number of cards were deemed as fairest player
